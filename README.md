# base16-vstheme
Base16 Ocean Theme for Visual Studio 2015, including ReSharper color support.

Supports the following file formats:
- C#
- Javascript
- HTML
- Razor
- CSS
- SCSS

## How to install

1. Select **Tools > Customize Colors**
2. Click **Import Theme**
3. Select the .vstheme file
